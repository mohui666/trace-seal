## Known limitations and remaining release gates

- Python, Renderer and Electron version metadata remains consistently at the latest published version, 0.2.0. The explicit release step must update it to 0.3.0 before tagging.
- The generated release candidate has not yet completed install/relaunch validation on a clean Windows VM.
- The installer is unsigned, so Windows SmartScreen may report an unknown publisher.
- Only Windows x64 is validated.
- TraceSeal still uses Python process-level hooks, copied-workspace sandboxing and transcript replay.
- HTTP cassette deliberately omits full request/response bodies, so it is not deterministic network replay.
- Rust Guard is a future Stage 4 scope item.
